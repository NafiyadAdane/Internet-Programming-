This are lists of the most common HTML tags, their attributes, and their functions, separated by categories. Basic Structure : Defines the document type and version of HTML. : The root element of an HTML document, and the lang attribute specifies the language. : Contains meta information and other elements that aren't displayed directly on the web page. : Specifies the character encoding for the document. : Sets the title of the web page, displayed in the browser's title bar or tab. <body>: Contains the content of the web page. Headings <h1> to <h6>: Define headings, where <h1> represents the largest heading and <h6> the smallest. Text Formatting <p>: Defines a paragraph. <br>: Inserts a line break. <hr>: Inserts a thematic break or horizontal rule. <strong>: Represents strong importance or emphasis. <em>: Represents stress emphasis. <mark>: Represents highlighted or marked text. <del>: Represents deleted text. <ins>: Represents inserted text. <sub>: Defines subscript text. <sup>: Defines superscript text. <pre>: Represents preformatted text. Links <a href="URL" target="_blank">: Creates a hyperlink, where the href attribute specifies the link's destination, and target="_blank" opens the link in a new tab. Images <img src="URL" alt="Description">: Embeds an image, where src specifies the image source, and alt provides a description for screen readers or when the image fails to load. Lists <ul>: Defines an unordered (bulleted) list. <ol>: Defines an ordered (numbered) list. <li>: Defines a list item. <dl>: Represents a description list. <dt>: Represents a term in a description list. <dd>: Represents a description of a term in a description list. Tables <table>: Defines a table. <tr>: Defines a table row. <th>: Defines a table header cell. <td>: Defines a table data cell. <caption>: Represents a table caption. <thead>: Groups the header content in a table. <tbody>: Groups the body content in a table. <tfoot>: Groups the footer content in a table. <colgroup>: Specifies a group of columns in a table. <col>: Represents a column within a table. Forms <form action="URL" method="GET/POST">: Creates an HTML form, where action specifies the URL to submit the form data, and method specifies the HTTP method (GET or POST). <input type="text/email/password/checkbox/radio">: Represents an input field in a form, where type specifies the type of input field. <textarea>: Represents a multiline text input field. <select>: Represents a drop-down list. <option>: Represents an option within a drop-down list. <button>: Represents a button. <label>: Represents a caption for a form element. Multimedia <audio src="URL" controls>: Embeds an audio player, where src specifies the audio source, and controls adds playback controls. <video src="URL" width="320" height="240" controls>: Embeds a video player, where src specifies the video source, width and height set the dimensions, and controls adds playback controls. <source src="URL" type="audio/mime-type">: Specifies multiple media resources for the <audio> or <video> elements, where src specifies the media source and type specifies the MIME type. <track kind="subtitles" src="URL" srclang="language" label="Label">: Adds text tracks (like subtitles) to the <audio> or <video> elements, where kind specifies the type of text track, src specifies the source, srclang specifies the language, and label provides a user-readable title. Sections and Layout <header>: Represents a container for introductory content or navigation links. <nav>: Represents a section with navigation links. <main>: Represents the main content of a document. <section>: Represents a generic section of a document. <article>: Represents a self-contained composition in a document, like a blog post or news article. <aside>: Represents content that is tangentially related to the content around it, often used for sidebars. <figure>: Represents any content referenced from the main content, like an image, diagram, or chart. <figcaption>: Represents a caption for a <figure> element. <footer>: Represents a footer for a section or page, typically containing meta-information, authorship, copyright, or contact details. Styles and Semantics <style>: Represents style information for a document, typically used to include CSS rules. <div>: Represents a generic container for grouping and applying styles to block-level content. <span>: Represents a generic container for grouping and applying styles to inline content. <class>: An attribute used to apply CSS styles to multiple elements with the same class name. <id>: An attribute used to apply CSS styles or JavaScript actions to a single unique element. Scripting <script src="URL">: Represents a script (e.g., JavaScript) to be executed in the HTML document, where src specifies the script source. Meta information <meta name="description" content="Description">: Provides metadata about the HTML document, like a description for search engines. <meta name="keywords" content="Keyword1, Keyword2">: Provides a list of keywords relevant to the HTML document for search engines. <meta name="viewport" content="width=device-width, initial-scale=1.0">: Sets the viewport width and initial zoom level for responsive web design. <meta http-equiv="refresh" content="30">: Sets an automatic refresh interval for the HTML document (in seconds). <link rel="stylesheet" href="URL">: Specifies an external style sheet to be linked to the HTML document. <base href="URL">: Specifies a base URL for all relative URLs in the HTML document.
